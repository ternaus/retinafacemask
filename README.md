# retinafacemask
Detector for faces with masks / no masks on top of them.

## Dataset
### Faces with masks:

https://github.com/X-zhangyang/Real-World-Masked-Face-Dataset

### Faces without masks
http://www.robots.ox.ac.uk/~vgg/data/vgg_face2/
http://shuoyang1213.me/WIDERFACE/
